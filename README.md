Sistema de Gestión de Paseadores
Aplicación desarrollada como proyecto académico en el curso de Programación 1, orientada a la gestión de clientes, paseadores y servicios de paseo de mascotas.
El sistema permite administrar contrataciones, validar condiciones de servicio y simular la operativa de una empresa de paseadores.

---

Tecnologías utilizadas
- Lenguaje de programación Java Script y CSS
- Lógica de programación estructurada
- Manejo de datos en memoria

---

Funcionalidades principales
- Registro de clientes, mascotas y paseadores
- Sistema de contratación de paseos
- Validación de reglas de negocio:
- Compatibilidad según tamaño de perros
- Control de cupos disponibles por paseador
- Cancelación de servicios
- Gestión de estado de contrataciones

---

Conceptos aplicados

- Estructuras de control (condicionales y bucles)
- Modularización del código
- Validación de datos
- Lógica de negocio
- Resolución de problemas

---

Objetivo del proyecto
Simular el funcionamiento de un sistema real de gestión de paseadores, aplicando conceptos fundamentales de programación y desarrollando lógica para resolver situaciones del mundo real.
